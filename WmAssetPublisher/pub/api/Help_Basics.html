Navigation Frame
To get started, click a topic in the Contents tab in the navigation frame on the left side of this browser window. The navigation frame contains the following three tabs:
n
Contents. Displays the contents of the Help system in the form of an expandable/collapsible tree view. Closed book icons represent entries that have subentries. Click a closed book to open it and see its contents. When you expand a book, the closed book icon is replaced by an open book icon. You can click the book again to close it. The page icons represent individual Help topics.
n
Index. Displays entries from the Help system index. Scroll to find the index entry you want and double-click the entry to display the corresponding Help topic, or select an index entry and then click Display.
n
Search. Allows you to search the full text of the Help system. Type the word or phrase to search for, and then press Enter or click Go. The Search tab displays a list of all the topics in the Help system that contain the word or phrase you entered. The topics found by search are ranked in order of relevance. The higher the ranking, the more likely the topic includes the word or phrase you searched for.
Buttons in the Help Toolbar
A toolbar frame appears above the topic frame of your help system. The following buttons are available:
 
This button...
Does this...
Show Navigation. Click to display the navigation frame on the left side of the browser window. Appears only when the window is open in topic-only mode.
Show in Contents. Click to show the position of the current topic page in the navigation frame. Appears only the navigation frame is visible.
Print. Click to print the current topic on your default printer.
Bookmark. Click to create a bookmark to this topic in your browser.
