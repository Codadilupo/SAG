Screen Heading: Asset Publisher > Publish > Published Metadata
Use the Latest Published Metadata screen to view details of the latest publish operation.
To open the Latest Published Metadata screen, click Asset Publisher under the Solutions menu on the Navigation panel. Select Publish and click Latest Publish on the Publish screen.
n
Click Return to Publish to return to the Publish screen.
Publish Details
This section displays information about the latest publish operation, including:
 
Field
Description
Type
Type of publish operation: complete or incremental.
Status
Operation status: executing, complete, error, etc.
Published Assets
Number of assets that have been published so far.
Total Assets
Total number of assets for which metadata will be published.
Duration
Length of time the publish operation has taken.
TN Assets
Whether Trading Network assets are included in this publish operation: true or false.
Packages
Packages included in the publish operation.
Action / Asset Name / Metadata Type
This section describes the actions performed on each asset.
 
Field
Description
Action
Whether the asset was added, changed, or deleted.
Asset Name
The name of the affected asset.
Metadata Type
The data type of the asset (package, folder, or flow service).
