Screen Heading: Asset Publisher > Retracted Assets
Use the Retracted Assets screen to view the details of the latest retract operation.
To open the Retracted Assets screen, click Retract under the Solutions menu on the Navigation panel. Click Latest Retract on the Retracted Assets screen.
n
Click the Return to Retract link to return to the Retract screen.
Retract Details
This section displays information about the latest retract operation, including:
 
Field
Description
Status
Operation status: executing, complete, error, etc.
Retracted Assets
Number of assets whose data has been removed from the library.
Duration
Length of time the retract operation took.
TN Assets
Whether Trading Network assets are included in this retract operation: true or false.
Packages
Names of packages included in the publish operation.
Action / Asset Name
This section describes the actions performed on each asset.
 
Field
Description
Action
Whether the asset was deleted, changed, etc.
Asset Name
The name of the affected asset.
