Screen Heading: Asset Publisher > Assets > Edit
Before you can publish metadata to the library, you must first use the Edit Assets screen to select (or deselect) the assets whose metadata you want to publish. Previously selected publishable assets are identified with a green check mark.
Note: Deselecting an asset does not delete that asset’s data from the Metadata Library. It simply removes it from the list of publishable assets, ensuring that it will not be included in the next publish operation. To delete an asset’s data from the library, see the Retract Screen.
To open the Edit Assets screen, click Asset Publisher under the Solutions menu on the Navigation panel. Select Asset Selection and click Edit Assets on the Asset Selection screen.

Save
Click Save to save changes to all fields. These changes will be reflected in the Selected Assets list the next time the Publish screen is displayed.
