Screen Heading: Asset Publisher > Configuration
Use the Configuration screen to configure the connection from the Integration Server to the Metadata Library Server.
Important! You must set up an account on the Metadata Library Server before you can use the Configuration screen to configure the connection. For more information, see the webMethods Metadata Library Configuration Guide.
To open the Configuration screen, click Asset Publisher under the Solutions menu on the Navigation panel.The Asset Publisher Configuration screen will be displayed by default.
n
Click Edit Configuration to edit the configuration settings.
n
Click Test Connection to test the connection between the Integration Server and the Metadata Library Server.
Metadata Library Configuration Settings
 
Parameter
Description
IS Host Name
IP address of this Integration Server, or a name assigned to it to identify it to Designer.
Protocol
Communication protocol, which is HTTP.
Host
Host name or IP address of the server that hosts the Metadata Library.
Port
Port number on which the Metadata Library Server listens for incoming requests from this IS.
User Name
Name of a user account on the Metadata Library Server. This user must be able to query, insert, and delete all data about this Integration Server in the Metadata Library Server. The default user name is “Developer”.
Password
Password for User Name. The default password is “develop”.
