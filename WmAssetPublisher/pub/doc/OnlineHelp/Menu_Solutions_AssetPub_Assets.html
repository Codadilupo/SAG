Screen Heading: Asset Publisher > Assets
Before you can publish metadata to the library, you must first identify publishable assets. A publishable asset is an asset for which you want to publish metadata. Use the Asset Selection screen to view all packages on the Integration Server.
To open the Asset Selection screen, click Asset Publisher under the Solutions menu on the Navigation panel. Click Asset Selection.
The Integration Server displays a list of the Trading Networks assets (Doc Types and Partner Profiles) and all the packages on the server. Packages that have been previously selected as publishable assets are identified with a green check mark.
n
Click Edit Assets to add or remove assets from the publishable assets list.
