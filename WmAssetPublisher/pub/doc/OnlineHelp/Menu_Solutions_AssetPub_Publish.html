Screen Heading: Asset Publisher > Publish
Use the Publish screen to publish (write metadata) to the Metadata Library for.
Note: Before you can publish, you must first identify the assets for which you want to publish metadata (publishable assets). Use the Edit Assets Screen to identify publishable assets.
To open the Publish screen, click Asset Publisher under the Solutions menu on the Navigation panel. Click Publish.
n
Click the Latest Publish link to view the details of the latest publish operation.
n
Click the Publish button to run the publish operation on all selected publishable assets.
Tip! A complete publish operation can consume substantial amounts of time and memory, so you should consider scheduling them to run at off-peak hours. For more information, read about the WmAssetPublisher.pub.asset:publish service in the webMethods Metadata Library Configuration Guide.
Publish Type
Select the type of publish operation you want to perform, incremental or complete.
Incremental
Publishes metadata only for selected assets that have changed since the last publish operation.
Complete
Publishes metadata for all selected assets, whether they have changed or not.
Selected Trading Networks Assets
Select the publishable assets that you want to include in the publish operation.
Doc Types, Partner Profiles
Trading Network assets that are publishable. Select the check box to include these in the current publish operation.
Selected Integration Server Packages
Integration Server packages that are publishable assets. Select the check box of each package you want to include in the current publish operation.
Select: All, None
Click All to select all the check boxes in the list. Click None to deselect all the check boxes.
Publish
Select Publish to start the publish operation for all checked assets. After the publish operation is complete, the results will be displayed on the Latest Published Metadata Screen by default.
