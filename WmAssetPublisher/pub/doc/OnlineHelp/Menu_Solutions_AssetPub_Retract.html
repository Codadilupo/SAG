Retract Screen

Screen Heading: Asset Publisher > Retract
Use the Retract screen to delete a publishable asset’s data from the Metadata Library.
Note: Retracting, or deleting, an asset’s data from the Metadata Library does not prevent the asset’s metadata from being published during the next publish operation. It simply removes the current data for that asset from the Metadata Library. To ensure that an asset is not be included in the next publish operation, see the Edit Assets Screen.
To open the Retract screen, click Retract under the Solutions menu on the Navigation panel.
n
Click the Latest Retract link to view the results of the latest retract operation.
n
Click the Retract button to remove data for all selected assets from the Metadata Library.
Published Trading Networks Assets
Doc Types, Partner Profiles
Trading Network assets whose metadata is currently included in the Metadata Library (that is, they have been previously published). Select the check box to remove this data.
Published Integration Server Packages
Integration Server packages whose metadata is currently included in the Metadata Library (that is, they have been previously published). Select the check box of each package whose data you want to remove from the library.
Select: All, None
Click All to select all the check boxes in the list. Click None to deselect all the check boxes.
Retract
Select Retract to start the retract operation for all checked assets. After the publish operation is complete, the results will be displayed on the Latest Retracted Assets Screen by default.
