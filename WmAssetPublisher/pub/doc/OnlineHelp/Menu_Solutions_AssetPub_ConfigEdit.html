Screen Heading: Asset Publisher > Configuration > Edit
Use the Edit Configuration screen to configure the connection from the Integration Server to the Metadata Library Server.
Important! You must set up an account on the Metadata Library Server before you can use the Configuration screen to configure the connection. For more information, see the webMethods Metadata Library Configuration Guide.
To open the Edit Configuration screen, click Asset Publisher under the Solutions menu on the Navigation panel. Click Edit Configuration on the Configuration screen.
n
Click Return to Configuration to return to the previous screen.
n
Click Save Changes to save any change to the connection.
