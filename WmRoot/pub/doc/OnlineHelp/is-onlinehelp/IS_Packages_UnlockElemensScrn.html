Unlock Elements Screen
Screen Heading: Packages > Management > Unlock Elements
Use the Unlock Elements screen to remove user locks from elements on the Integration Server.
To open the Unlock Elements screen, open the Packages Management Screen and click View Locked Elements. On the View Locked Elements Screen, click Unlock Elements.
Important:  
Exercise caution when you remove user locks, since a user can lose changes. If you unlock an element while a user is editing it, the user will not know until he or she tries to save changes to the element and the operation fails. Always confirm with the user before removing his or her lock on an element.
Click Return to Locked Elements to return to the View Locked Elements Screen.
Locked by System
Lists elements whose corresponding files are marked read-only on the server file system. You cannot remove a system lock via the Integration Server Administrator. On the server's file system, you must make the element's files writable and reload the package.
Locked by Current User (username)
Lists elements that are locked by you, the server administrator (or the username with which you logged into the Integration Server Administrator). Before you unlock an item, make sure that you have saved all changes to the server.
Locked by Other Users
Lists elements that are locked by other users on the server. Before you remove a user's lock, make sure that the user has saved all changes to the server. If not, the user will lose all changes that they made to the element since they last saved it to the server.
Unlock Selected Elements
Click to unlock the elements with selected check boxes in the list. Make sure that you inform users before removing their locks.
Note:  
If you receive an "element failed to unlock" message after clicking Unlock Selected Elements, then you may have elements listed that do not have corresponding files on the Integration Server (in other words, the files were deleted from the server). To update the view of locked elements in the Integration Server Administrator, use the Sync to Name Space command on the previous screen.
