Edit JMS Global Trigger Controls Screen
Screen Heading: Settings > Messaging > JMS Settings > Global JMS Trigger Controls > Edit
Use the Global JMS Trigger Controls screen to change the percentage of the server thread pool that can be used by JMS triggers and to adjust the threads used for processing concurrent JMS triggers.
To open the JMS Trigger Management screen, click Messaging under the Settings menu on the Navigation panel. Under JMS Configuration on the Messaging Screen, select JMS Trigger Management. Click Edit JMS Global Trigger Controls.
Click Return to JMS Trigger Management to return to the previous screen.
Global JMS Trigger Controls
Use this section to adjust thread usage for JMS triggers.
Thread Pool Throttle
Specify the maximum percentage of the server thread pool that can be used for JMS triggers. This includes thread used to retrieve messages from the JMS provider and threads used to process messages. You must enter a value greater than zero.
Individual Trigger Processing Throttle
Specify the percentage of the configured maximum execution threads value at which you want to the server to function. Integration Server applies this percentage to the maximum execution threads value for all concurrent JMS triggers.
Save Changes
Saves the specified settings. Server restart is not required for these changes to take effect. Any changes you make are persisted across server restarts.
