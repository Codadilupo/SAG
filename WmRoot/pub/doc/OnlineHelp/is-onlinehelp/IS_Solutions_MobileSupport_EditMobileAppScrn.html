Edit Mobile Application Screen
Screen Heading: Mobile Support > Mobile Applications > Edit Mobile Application
Use the Edit Mobile Application screen to edit the association between a mobile application and one or more mobile sync components.
To open the Edit Mobile Application screen, click Mobile Support under the Solutions menu on the Navigation panel, and then click Mobile Applications. Click the name of the mobile application you want to edit on the Mobile Applications screen.
Click Return to Mobile Applications to return to the previous screen.
Click the Save Changes button to save changes you made.
