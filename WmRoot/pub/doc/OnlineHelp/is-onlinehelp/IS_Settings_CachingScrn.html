Screen Heading: Settings > Caching
Use the Caching screen to add and edit cache managers using Integration Server Administrator.
To open the Caching screen, click Caching under the Settings menu on the Navigation panel.
Click Add Cache Manager to create a cache manager.
Under System Cache Managers, Integration Server Administrator displays the system cache managers and are identified by names that begin with SoftwareAG.
Public Cache Managers
Name
Name of the cache manager.
Reload
Reloads the cache manager without restarting Integration Server Administrator.
Start/Shut down
Shuts down or restarts a cache manager.
Delete
Deletes a cache manager.
