Associate Scopes to Clients Screen
Screen Heading: Security > OAuth > Scope Management > Associate Scopes to Clients
Use the Associate Scopes to Clients screen manage scope and client associations. The Associate Scopes to Clients screen is divided into two areas:
You can associate scopes to clients in the Scopes area (on the left). Use the Scopes area to make associations when you want to associate multiple clients to a single scope. This offers the advantage of adding or removing several associated clients for the scope at one time.
You can associate clients to scopes in the Clients area (on the right). Use the Clients area to make associations when you want to associate multiple scopes to a single client. This offers the advantage of adding or removing several associated scopes for the client at one time.
To open the Associate Scopes to Clients screen, go to the OAuth Screen and click Scope Management. Then click Associate Scopes to Clients.
Click Return to Scope Management to return to the Scope Management Screen.
Scopes
Use this section to select a scope and then specify the clients associated to it.
Select Scope
Select the scope to which you want to add an association to one or more clients.
Clients associated with Scope
Specifies the clients associated with the scope.
To remove clients from the scope, select (highlight) the client or clients that you want to remove and use the -> button to move clients to the Remaining Clients list.
Remaining Clients
Specifies the clients available to associate with the scope.
To add clients to the scope, select (highlight) the client or clients that you want to add and use the <- button to move clients to the Clients associated with Scope list.
Clients
Use this section to select a client and then specify the scopes associated to it.
Select Client
Select the client to which you want to add an association to one or more scopes.
Scopes in this Client
Specifies the scopes associated with the client.
To remove scopes from the client, select (highlight) the scope or scopes that you want to remove and use the -> button to move clients to the Remaining Scopes list.
Remaining Scopes
Specifies the scopes available to associate with the client.
To add scopes to the client, select (highlight) the scope or scopes that you want to add and use the <- button to move scopes to the Scopes in this Client list.
Save Changes
Updates Integration Server with the new or changed scope association settings. Your changes take effect immediately.
