Menus and Screens : Integration Cloud Menu : Settings Screen
Settings Screen
Screen Heading:Integration Cloud > Settings
Use the Settings screen to specify information about the Integration Cloud server.
To open the Settings screen, click Settings under the Integration Cloud menu on the Navigation pane.
Settings
User Name
The user name the on-premise Integration Server must use to access Integration Cloud.
Password
The password the on-premise Integration Server must use to access Integration Cloud.
webMethods Integration Cloud URL
The URL of the Integration Cloud server.
Update Settings
Click to save the settings.
