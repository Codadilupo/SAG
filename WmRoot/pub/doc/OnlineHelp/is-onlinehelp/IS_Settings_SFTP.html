Screen Heading: Settings > SFTP
Use the SFTP screen to view the list of SFTP server alias definitions in webMethods Integration Server.
The SSH File Transfer Protocol (SFTP) is a network protocol that is based on the Secure Shell protocol (SSH). SFTP facilitates secure file access, file transfer, and file management over any reliable data stream.
To open the SFTP screen, click SFTP under the Settings menu on the Navigation panel.
Click Create Server Alias to create new SFTP server aliases.
Click User Alias Settings to view the SFTP user alias definitions and to create new SFTP user aliases.
SFTP Server List
Displays the SFTP server aliases that are defined in your Integration Server.
Alias
Displays the alias name.
Host
Displays the host name or IP address of the SFTP server represented by this alias.
Port
Displays the port number that is used by the SFTP server represented by this alias.
Proxy Alias
Displays the proxy alias through which the request is to be routed. The proxy alias can be HTTP, HTTPS, or SOCKS.
Host Key Fingerprint
Displays the public key of the SFTP server represented by this alias.
Delete
Deletes the SFTP server alias definition from Integration Server.
