Edit Scope Screen
Screen Heading: Security > OAuth > Scope Management > scopeName > Edit
Use the Edit Scope screen to modify existing scopes.
To open the Edit Scope screen, go to the OAuth Screen and click Scope Management. Under Defined Scopes, click the name of the scope in the Name column.
Click Return to Scope Management to return to the Scope Management Screen.
Scope Configuration
Name
Specifies a unique name for the scope. You cannot change the Name field for a scope.
Folders and services
Specifies the list of folders and individual services that the client can access on behalf of the resource owner. You can add more than one folder or service at a time by specifying multiple lines, one folder or service to a line. Press ENTER to separate lines.
Note:  
Integration Server validates the entries. If you enter a folder or service that does not exist, Integration Server does not add the scope.
Save Changes
Updates Integration Server with the new scope settings. Your changes take effect immediately.
