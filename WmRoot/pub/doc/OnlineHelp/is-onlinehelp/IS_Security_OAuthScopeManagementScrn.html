Scope Management Screen
Screen Heading: Security > OAuth > Scope Management
Use the Scope Management screen to view and delete scopes.
To open the Scope Management screen, go to the OAuth Screen and click Scope Management.
Click Return to OAuth to return to the OAuth Screen.
Click Add Scope to go to the Add Scope Screen.
Click Associate Scopes to Clients to go to the Associate Scopes to Clients Screen.
Defined Scopes
Name
Specifies the name of the scope. Click the name to go to the Edit Scope Screen.
Folders and services
Specifies the folders and individual services that the client can access on behalf of the resource owner.
Delete
Deletes the scope specified in the Name column. To delete a scope, click the icon in the Delete column.
