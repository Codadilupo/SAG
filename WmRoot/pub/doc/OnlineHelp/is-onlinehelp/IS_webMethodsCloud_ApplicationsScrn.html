Screen Heading:Integration Cloud > Applications
Use the Applications screen to view, upload, and delete Integration Cloud applications.
To open the Applications screen, click Applications under the Integration Cloud menu on the Navigation pane.
Click Define Integration Cloud Application to define an application.
Integration Cloud Applications
This section lists the applications that exist on the on-premise Integration Server and can be shared with Integration Cloud.
Name
A unique name for the application.
Description
Description of the application.
Last Uploaded Time
The last time the application was uploaded to Integration Cloud.
Upload
Click one of the following to upload applications to Integration Cloud:
