Edit SFTP Server Alias Properties Screen
Screen Heading: Settings > SFTP > SFTP Server Alias Name > Edit
Use the Edit SFTP Server Alias Properties screen to edit an alias that has been defined for an SFTP server.
You can edit only the Host Name or IP Address, Port Number, and Host Key Location properties. You cannot leave the Host Name or IP Address and Port Number fields empty.
To open the Edit SFTP Server Alias screen, click SFTP under the Settings menu on the Navigation panel. In the SFTP Server List, click the name of the SFTP server alias that you want to edit. Integration Server displays the properties screen for that alias.
Click Return to SFTP to return to the list of SFTP server aliases defined in your Integration Server.
SFTP Server Alias Properties
Host Name or IP Address
Update the host name or IP address of the SFTP server. An SFTP server alias name:
Can contain only underscores (_) and periods (.) as special characters.
Cannot begin with the string "http://".
Can be of a maximum length of 255 characters.
Port Number
Update the port number of the SFTP server. The port number must be within the range of 0 and 65535 (inclusive).
Preferred Key Exchange Algorithm
Update your choice of algorithms that Integration Server presents to the SFTP server for key exchange. You can specify the order in which Integration Server presents the algorithms to the SFTP server by clicking Up or Down. The SFTP server has its own set of preferred algorithms configured. At the time of key exchange, one of the algorithms supported by both Integration Server and the SFTP server will be chosen.
Proxy Alias
Specify the proxy alias through which the request is to be routed. The proxy alias can be HTTP, HTTPS, or SOCKS. If a proxy alias is not specified, Integration Server makes outbound requests using each enabled proxy server alias until the request is sent successfully or all proxy servers have been tried.
Host Key Location
In case you have modified the SFTP server details, update the location of the public key of the updated SFTP server.
Important:  
The public key file must be present on the same machine in which you have installed Integration Server.
Host Key Fingerprint
Displays the public key fingerprint of the SFTP server that was previously defined.
Get Host Key
If you do not have the public key of the SFTP server, click Get Host Key. Integration Server will retrieve the public Key for the host and port you specified and save it in a temporary folder. Integration Server will then display the path to the retrieved public key file in the Host Key Location field.
Save Changes
Updates the SFTP server alias definition.
