Global Variables Screen
 
Add Global Variables Screen
Edit Global Variable Screen
Screen Heading: Settings > Global Variables
Use the Global Variables screen to create, view, edit, or delete global variables. Global variables are key/value pairs that you can use in flow services. Instead of hard coding the values for variables at the time of execution of a flow service, you can create global variables in Integration Server Administrator. At run time, Integration Server substitutes the variable with the value assigned to it.
To open the Global Variables screen, click Global Variables under the Settings menu on the Navigation panel.
Click Add Global Variable to create a new global variable.
Click the underlined name of a global variable to view or change details about a global variable.
Key
A name for the global variable. Integration Server uses this key to refer to the global variable while performing global variable substitution.
Global variable names:
Cannot include any special characters except period (.) and underscore (_).
Can be of a maximum length of 255 characters.
Are case-sensitive.
Must be unique.
Value
Value for the global variable.
Is Password?
Indicates whether the global variable you are defining is a password.
Yes indicates that the global variable is a password. In case of passwords, Integration Server encrypts and saves the text that you enter in the Value field as an outbound password.
No indicates that the global variable is not a password.
Delete
Deletes the global variable.
