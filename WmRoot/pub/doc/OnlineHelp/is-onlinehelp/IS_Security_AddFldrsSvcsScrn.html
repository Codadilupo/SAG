Add Folders and Services Screen
Screen Heading: Security > Ports > Edit Access Mode > portName
Use the Add Folders and Services screen to edit the list of folders and services that are either "allowed" or "denied" (depending on the current Access Mode setting).
To open the Add Folders and Services screen, go to the Edit Access Mode Screen and select Add Folders and Services to Allow List.
Click Return to Edit Access Mode to return to the current access mode settings.
Add Folders and Services to Allow List
Use this section to compose the list of services and/or folders that are allowed for this port.
Enter one folder or service per line
Specifies the names of services and/or folders that you want to allow. To add a name to the list, type a fully qualified folder name or service name in the text box on a line by itself. You may use the following wildcard characters to match a group of folders or services with similar names:
Character
Purpose
Example
*
Matches any sequence of characters
orders.inbound:SGX*
?
Matches any single character
orders.inbound:post???Order
Select a set of folders and services
Use this section to select a group of services and/or folders that have the same ACL setting. After you make your selection, you can append the results to the allow list.
Select an ACL
Lists the ACLs that are defined on the server. To select a group of services and/or folders with the same Execute ACL, select the name of the ACL from this list.
Append Selected
Appends the selected names to the allow list.
Save Additions
Updates the allow list. Your changes take effect immediately.
Clear Entries
Removes all entries from the allow list.
