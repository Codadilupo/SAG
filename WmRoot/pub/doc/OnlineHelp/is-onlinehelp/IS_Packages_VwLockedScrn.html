View Locked Elements Screen
 
Lock Details Screen
Unlock Elements Screen
Screen Heading: Packages > Management > Locked Elements
Use the View Locked Elements screen to show a list of user and system locked elements on the webMethods Integration Server and unlock elements.
To open the View Locked Elements screen, open the Packages Management Screen and click View Locked Elements.
Click Return to Package Management to return to the main package management screen.
Click Unlock Elements to unlock elements that are locked by you or other users on the Integration Server. Make sure that you inform users before removing their locks.
Click Sync to Name Space if there are locked elements shown that do not have corresponding files on the Integration Server (in other words, the files were deleted from the server). This command updates the Integration Server Administrator's view of unlocked elements by polling the Integration Server and updating the list. Note that this command is automatically run when the Integration Server is started or restarted.
Locked by System
Lists elements whose corresponding files are marked read-only on the Integration Server file system. You cannot remove a system lock via the Integration Server Administrator. On the server's file system, you must make the element's files writable and reload the package.
Locked by Current User (username)
Lists elements that are locked by you, the server administrator (or the username with which you logged into the Integration Server Administrator). Before you unlock an item, make sure that you have saved all changes to the server.
Locked by Other Users
Lists elements that are locked by other users on the server. Before you remove a user's lock, make sure that the user has saved all changes to the server. If not, the user will lose all changes that they made to the element since they last saved it to the server.
