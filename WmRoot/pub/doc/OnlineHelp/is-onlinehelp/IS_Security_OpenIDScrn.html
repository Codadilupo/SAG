Menus and Screens : Security Menu : OpenID Screen
OpenID Screen
Screen Heading: Security > OpenID
Use the OpenID screen to view a list of OpenID Providers used by Integration Server.
To open the OpenID screen, click OpenID under the Security menu on the Navigation panel.
Click Reload OpenID Providers to refresh the list of OpenID Providers.
OpenID Provider List
Name
Name of the OpenID Provider.
Issuer
A URI that uniquely identifies the OpenID Provider.
Client ID
An identifier that the OpenID Provider uses to identify the client application in Integration Server. This value is assigned to your application when you register your application with the OpenID Provider. A value of “0” means that client_id is not specified in the provider configuration file.
Delete Provider
Click to delete the OpenID Provider.
Note:  
If you delete salesforce.com from the OpenID Provider List, Integration Server recreates it upon startup.
