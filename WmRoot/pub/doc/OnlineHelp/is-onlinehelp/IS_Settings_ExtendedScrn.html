Extended Settings Screen
 
Show and Hide Keys Screen
Screen Heading: Settings > Extended
Use the Extended Settings screen to view and/or edit your server's advanced parameter settings. These parameters affect the operation of your server. You should not need to change these settings unless requested to do so by Software AG Global Support.
To open the Extended Settings screen, click Extended under the Settings menu on the Navigation panel.
Click Edit Extended Settings to change the parameter settings shown on this screen.
Click Show and Hide Keys to specify which keys (parameters) are displayed on this screen.
Extended Settings
Displays selected server parameters. Note that this screen displays only those parameters that you enable (make visible) on the Show/Hide Key screen. For descriptions of the server parameters, see "Server Configuration Parameters" in the webMethods Integration Server Administrator’s Guide.
Note:  
Use care when editing these parameter settings. The editor makes no effort to ensure that your changes are valid. Whatever you type in this text box is saved in the server configuration file (server.cnf).
Save Changes
Edit Mode only. Saves the parameter settings shown in the Extended Settings text box.
