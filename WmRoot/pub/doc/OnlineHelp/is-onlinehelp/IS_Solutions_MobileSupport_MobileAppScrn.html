Mobile Applications Screen
Screen Heading: Mobile Support > Mobile Applications
Use the Mobile Applications screen to associate a mobile application with one or more mobile sync components, to view a list of the mobile sync components that a mobile application is associated with, and to change or remove the associations between a mobile application and mobile sync components.
Click Add Mobile Application to associate a mobile application with a mobile sync component.
Mobile Applications
Application Name
Lists the names of the mobile applications that are associated with mobile sync components. Click an application name to edit the association between the application and the mobile sync components.
Application Version
Indicates the version of the mobile application.
Mobile Sync Component Aliases
Indicates the mobile sync components that are associated with the mobile application.
Delete
Click the button in this field to delete the application from the list and remove its associations with the specified mobile sync components.
