Define Application Screen
Screen Heading:Integration Cloud > Applications > Define Application
Use the Define Application screen to create Integration Cloud applications.
To open the Define Application screen, click Applications under the Integration Cloud menu on the Navigation pane and then click Define Integration Cloud Application.
Click Return to Integration Cloud Applications to return to the list of applications.
Integration Cloud Application
Name
A unique name for the application.
Description
Description of the application.
Assign Services to Application
For each package, specify the services to share with Integration Cloud as follows:
1. Under Package/Services, click  to expand the services available in the package.
2. Select each service you want to expose to Integration Cloud.
3. Under Display Name, specify the name of the service to display in Integration Cloud or accept the default.
Save Changes
Click to save the application.
