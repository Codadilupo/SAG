Add and Remove Users Screen
Screen Heading: Security > User Management > Add and Remove Users
Use the Add and Remove Users screen to add a new user account to your server or delete an existing one. To add or remove OpenID Provider users, see Add and Remove OpenID Provider Users Screen.
To open the Add and Remove Users screen, go to the User Management Screen and click Add and Remove Users.
Click Return to User Management to return to the list of groups and users that are currently defined on your server.
Create Users
Use this section to define a new user account.
User Names
Specifies the name of the user account that you want to add. Type a unique name using any combination of letters, numbers or symbols. If you want to add more than one new user name, type each name on a separate line.
Important:  
User names on the webMethods Integration Server are case sensitive (for example, JDScott and jdscott are two distinct IDs). When you create a user name, be sure to type it exactly as you want the client to enter it.
Password
Specifies the password for this account. When using basic authentication (user names and passwords), the server uses the password to authenticate clients that log on with this user account. Type a string of characters using any combination of letters, numbers or symbols. If you have specified multiple names in the User Names list, this password will be assigned to all of them.
Important:  
Passwords on the webMethods Integration Server are case sensitive (for example, Green and green are two distinct passwords). When you create a password, be sure to type it exactly as you want the client to enter it.
Re-Enter Password
Confirms that the password has been entered correctly. Retype the password that you specified in Password.
Allow Digest Authentication
Specifies whether to use password digest as an authentication option.
Create User
Creates the user accounts that you have specified. These accounts take effect immediately.
Use this section to delete a user account from your server.
Select Users
Lists the user accounts that are defined on your server. To delete a user account, click the check box next to its name. Note that you cannot delete the system accounts.
Remove Users
Deletes the user accounts that you have selected in the Select Users list.
