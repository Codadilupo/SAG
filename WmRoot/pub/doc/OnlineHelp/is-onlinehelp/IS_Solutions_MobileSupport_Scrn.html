The webMethods Mobile Support feature (provided in the WmMobileSupport package) is an enhancement to webMethods Enterprise Gateway. The Mobile Support feature facilitates data synchronization between mobile devices and a backend application.
To open the Mobile Support screens, click Mobile Support under the Solutions menu on the Navigation panel.
Click Mobile Sync Components on the Navigation panel to add, edit, enable, disable, suspend, and delete mobile sync components.
Click Mobile Applications on the Navigation panel to associate mobile applications with mobile sync components.
