Metadata Configuration Screen
 
Edit Metadata Configuration
Screen Heading: Asset Publisher > Configuration
Use the Asset Publisher Configuration screen to view CentraSite connection settings and to test your connection to CentraSite.
To open the Asset Publisher Configuration screen, click Metadata under the Settings menu on the Navigation panel.
Click Edit Configuration to change the configuration settings shown on this screen.
Click Test Connection to test the connection between Integration Server and CentraSite.
IS Identifier
The name or the IP address of the machine on which this Integration Server is running. By default, Integration Server populates this field with the IP address of the machine. You can, however, enter a name here, which Integration Server will treat as an alias. In addition, Integration Server includes this name with the published metadata. You can use any name, but it cannot include the following illegal characters:
#-&@^!%*:$./\\`;,~+=)(|}{][><".
CentraSite URL
The CentraSite URL to which to publish metadata about Integration Server assets.
User Name
The name of the user account on CentraSite that will be used for publishing and retracting metadata.
Password
Password for User Name.
Test Connection
Tests the connection to CentraSite. Integration Server will display a status line at the top of the screen, indicating whether or not the connection is successful.
